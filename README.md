# StudyPRN Study Portal

Interactive study games for students in healthcare, built with HTML, CSS, and JavaScript.

## Games Available

- **Body Systems & Homeostasis** — explore organs, homeostasis, and diseases
- **IV Fluid Mastery** — tonicity, indications, and nursing cautions
- **Electrolyte Imbalances** — Na, K, Ca, Mg hyper and hypo states
- **Cranial Nerves**  - assessment, mnemonics, quizzes 

## How to Use

Open `index.html` in any browser, or visit the live GitHub Pages site.

HTML · CSS · JavaScript
